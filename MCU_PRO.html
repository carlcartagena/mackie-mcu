MACKIE – MCU PRO

WYSIWYG

Automation, Utilities (for now:undo/redo(shift), cancel, enter), Bank </>, Channel </>, Mixer Section, Transport Section, Views

VPOT ASSIGN

Switch between Volume/Pan/Send/Device mode.
Press the Send button multiple times to scroll through the send pages.
Press the Device button to toggle between parameter page and presets page.
Use EQ/INSTRUMENT buttons to browse through parameter pages.

VPOTS

control the parameters of the active mode. In volume mode, only the display changes, vpots are still mapped to pan.

UP/DOWN/LEFT/RIGHT

Same as arrow keys on computer keyboard.
When shift pressed: focus panel in the pressed direction in the application GUI.

FUNCTION BUTTONS

In Send Mode: select send page.
In Device Parameter Mode: select parameter page.
In Device Preset Mode: 1/2: previous/next preset, 3/4: previous/next preset bank, 5/6 previous/next preset creator.

FADER BANKS

Flip: flips the Vpots and faders control targets

MODIFIERS

Shift

Undo/Redo + shift-> redo.
FineTune for Vpots (or just click a vpot while moving it).

VIEWS

V1: Show Note/Audio Editor
V2: Show Automation Editor
V3: Show Device Chain
V4: Show Mixer
V8: Show Browser

DISPLAY

The display shows the transport position in BARS:BEATS:SUBDIVISION with tempo shown in the TICKS field.

NAVIGATION

Dial Change the transport position (no effect when playing). With shift pressed change the tempo.

AUTOMATION

Group Abort automation override.
Trim Write automation to clip launcher.
